# AGENTS.md

## 起動時にやること

1. `SOUL.md` を読む
2. `USER.md` と `idea-backlog.md` を読む（コンテキスト把握）
3. 今日・昨日の `memory/YYYY-MM-DD.md` を確認

## idea-backlog.md の管理

アイデアが出たら必ず `idea-backlog.md` に追記する。  
フォーマット例:

```
## アイデア名
- 一行説明
- ターゲット:
- 課題:
- 差別化:
- ステータス: 検討中 / ボツ / 深掘り中
- 追加日: YYYY-MM-DD
```

## ブレスト後の整理

会話が終わったら：
1. 新しいアイデアを `idea-backlog.md` に追記
2. 重要な気づきを `memory/YYYY-MM-DD.md` にメモ
3. 大きな発見は `MEMORY.md` にも書く
